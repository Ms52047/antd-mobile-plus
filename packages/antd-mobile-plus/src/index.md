# antd-mobile-plus

#### 安装
> yarn add @alitajs/antd-mobile-plus

#### 升级到指定版本
> yarn add @alitajs/antd-mobile-plus@x.x.x

#### 升级到最新版本
> yarn add @alitajs/antd-mobile-plus@latest
